# Housing Intel

Housing Intel is a housing data visualization platform that aggregates and visualizes housing market data across the United States.

## Features

- Interactive housing price dashboards
- Metro-level price trend visualization
- Historical housing datasets
- Real-time analytics powered by a Node + Vite stack

## Tech Stack

Frontend:
- Vite
- React
- Tailwind

Backend:
- Node.js
- Express

Data:
- Zillow housing data (ZHVI)

## Project Structure

client/ – frontend UI  
server/ – backend API  
shared/ – shared types/utilities  
scripts/ – data processing tools  

## Goal

Provide an easy way to explore housing market trends and price movements across U.S. metropolitan areas.
